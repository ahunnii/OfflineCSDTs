Sound Blocks

  • Name: Change Balance By Block
  • Type: Command
  • Description: This block adds the input parameter to the current balance value.
  • Name: Change Tempo By Block
  • Type: Command
  • Description: This Block adds the input parameter to the current Tempo value.
  • Name: Play Frequency HZ Block
  • Type: Command
  • Description: This block sets the current frequency value to the input parameter, given in Hertz.
  • Name: Play Note For Beats Block
  • Type: Command
  • Description: This Block plays a note for the input parameter number of beats.
  • Name: Play Sound Block
  • Type: Command
  • Description: This block plays a sound from the Sounds Library (similar to 'switch costume').
  • Name: Play Sound At Hertz Block
  • Type: Command
  • Description: This block plays the specified sound in the drop down box at the specified number of Hertz in the second drop down box.
  • Name: Rest For Beats Block
  • Type: Command
  • Description: Causes the current sound to not be played for the number of beats specified in the input parameter.
  • Name:Set Balance To Block
  • Type: Command
  • Description: This block sets the current balance value to the input parameter.
  • Name: Set Instrument To Block
  • Type: Command
  • Description: This block sets the current instrument to its corresponding number in the drop down. The number (1) corresponds to "sine", (2) "square", (3) "sawtooth", and (4) "triangle".
  • Name: Set Tempo To BPM Block
  • Type: Command
  • Description: This block sets the current temp in beats per minute to the value in the input parameter.
  • Name: Set Volume To Block
  • Type: Command
  • Description: This block sets the current volume percent value to the input parameter.
  • Name: Stop All Sounds Block
  • Type: Command
  • Description: This block stops all playing sounds.
  • Name: Stope Frequency Block
  • Type: Command
  • Description: This Block stops current Frequency.
