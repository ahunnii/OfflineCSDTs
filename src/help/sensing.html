Sensing Blocks

  • Name:
  • Type: Logical Operator
  • Description:
  • Name: Touching Sprite Block
  • Type: Logical Operator
  • Description: Returns true if you are touching a specific sprite, edge, or mouse.
  • Name: Touching Color Block
  • Type: Logical Operator
  • Description: Returns true if you are touching a specific color. Click the color, and then move your mouse over the color you want to detect.
  • Name: Ask and Wait Block
  • Type: Command
  • Description: Have the object ask a question and wait for the user to type a response.
  • Name: Answer Block
  • Type: Reporter
  • Description: This block reports the value of what was typed using the Ask and Wait Block.
  • Name: Mouse X Block
  • Type: Reporter
  • Description: This block returns the current X position of the mouse.
  • Name: Mouse Y Block
  • Type: Reporter
  • Description: This block returns the current Y position of the mouse.
  • Name: Reset Timer Block
  • Type: Command
  • Description: This block resets the timer.
  • Name: Timer
  • Type: Reporter
  • Description: This block returns the current value of the Timer.
  • Name: Attribute Of Block
  • Type: Reporter
  • Description: This block returns attributes of an object. These include s position, y position, direction, costume #, costume name size, width, height, left, right, top, bottom, volume, and balance.
  • Name: My Block
  • Type: Reporter
  • Description: This block returns attributes of an object. These include neighbors, self, other sprites, clones, other clones, parts, anchor, stage, children, parent, temporary?, name, costume, costumes, sounds, dangling? draggable?, width, height, left, right, top, bottom, rotation style, rotation x, rotation y, center x, center y.
  • Name: Object Block
  • Type: Reporter
  • Description: This block returns a reference to an object accessible through the drop down.
  • Name: URL Block
  • Type: Reporter
  • Description: Returns the input block text as a URL.
  • Name: Microphone Block
  • Type: Reporter
  • Description: This block returns attributes of the microphone object, including volume, note, frequency, samples, sample rate, spectrum, resolution.
  • Name: Video On Block
  • Type: Reporter
  • Description: This block returns video related attributes. Arguments in the first drop Down include snap, motion, and direction. The second drop down includes all named objects.
  • Name: Set Video Transparency Block
  • Type: Command
  • Description: This block sets the video transparency value provided by the input oval.
  • Name: Set Video Capture To Block
  • Type: Command
  • Description: This block takes a Boolean argument and applies it to set turbo mode, flat line ends, log pen vectors, video capture, and mirror video.
  • Name: Date Block
  • Type: Reporter
  • Description: This block returns the current system date.
