Looks Blocks

  • Name: Change Color Effect by Block
  • Type: Command
  • Description: This block adds the input parameter to the current color effect value. Drop down includes color, saturation, brightness, ghost, fisheye, whirl, pixelate, mosaic, and negative effects.
  • Name: Chang Size by Block
  • Type: Command
  • Description: This block adds the input parameter to the current size value in percent.
  • Name: Clear Graphic Effects Block
  • Type: Command
  • Description: This block resets graphic effects to the system default values.
  • Name: Flip Horizontal Block
  • Type: Command
  • Description: This block flips the sprite image on the horizontal axis.
  • Name: Flip Vertical
  • Type: Command
  • Description: This block flips the sprite on its vertical axis.
  • Name: Go Back Layers Block
  • Type: Command
  • Description: This block changes the active layer by a relative amount from the input parameter.
  • Name: Go To Front Layer Block
  • Type: Command
  • Description: This block changes the active layer to be the top layer. The drop down choices include both front and back layers.
  • Name: Hide Block
  • Type: Command
  • Description: This block makes a sprite invisible.
  • Name: Next Costume Block
  • Type: Command
  • Description: This block iterates through the costume library, which is available from the Costumes tab.
  • Name: Reflect Across X Axis Block
  • Type: Command
  • Description: This block reflects the sprite image relative to the X axis.
  • Name: Reflect Across Y Axis Block
  • Type: Command
  • Description: This block reflects the sprite image relative to the Y axis.
  • Name: Scale by Factor
  • Type: Command
  • Description: This block scales the costume by a percentage of the current size. Use this block to change the image size on the stage bigger or smaller.
  • Name: Switch To Costume Block
  • Type: Command
  • Description: This block changes a sprites current costume to the costume chosen in the drop down input.
  • Name: Set Color Effect To Block
  • Type: Command
  • Description: This block sets the current color to the input value. To know what color corresponds to what number, check out the Colors help page.
  • Name:Set Size To Block
  • Type: Command
  • Description: This block sets the current size of a sprite relative to it's full size. The input parameter is a percentage. To make a sprite larger than its default size, use a value greater than 100, to make it smaller, use a value less than 100.
  • Name: Show Block
  • Type: Command
  • Description: This block makes a sprite visible. Use it to un-hide a sprite.
